WordPress技巧之php生成密码

2015-06-15
技术研发

今天也挺奇葩的，不知道是忘记了博客密码还是数据库出现问题了，就是登录不上博客。那么就想到直接在数据库修改密码，那么出现个问题就是如何得到WP的密码加密方式，下面一段代码是生成一段加密的WP密码，直接把wp_users这个表对应用户的密码修改成下面生成的密码即可，mysql的命令操作不是本文重点，就不多介绍了。

<?php
$password = '123456';
global $wp_hasher;
if ( empty($wp_hasher) ) {
require_once('./wp-includes/class-phpass.php');//wordpress目录下的文件
$wp_hasher = new PasswordHash(8, TRUE);
}
echo $wp_hasher->HashPassword($password);
?>

Mac技巧之更新svn版本

2015-03-20
技术研发
  • 下载想要subversion更新的版本：http://www.wandisco.com/subversion/download#osx
  • 然后进行傻瓜式安装，安装完成后进入/opt/subversion目录查看文件。

  • 删除本地的低版本svn，需要获取root权限

    rm -rf /usr/bin/svn*

设置环境变量

sudo ln -s /opt/subversion/bin/svn* /usr/bin/
